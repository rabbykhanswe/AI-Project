How to Run this Project
Train the Brain: Open your terminal in the project folder and run:

Bash

python train_model.py
You will see "Model and Vectorizer saved." This creates the .pkl files.

Start the Website: Run:

Bash

python app.py
You will see something like Running on http://127.0.0.1:5000.





Test It:

Open your browser and go to http://127.0.0.1:5000.

Type "Win a free iPhone now" -> Click Check -> Result: SPAM


Type "Let's meet for lunch" -> Click Check -> Result: Not Spam (Safe)
